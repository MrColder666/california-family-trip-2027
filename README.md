## 📖 项目简介

这是一个专为两大两小家庭设计的加州15天旅游规划网站，提供完整的行程安排、预算计算和实用指南。通过现代化的交互设计和瑞士极简风格，帮助家庭轻松规划一次难忘的加州之旅。

**项目特色：**
- 💰 精确预算控制：严格控制在7万人民币以内
- 📅 详细行程规划：15天完整行程安排
- 🏙️ 多城市覆盖：洛杉矶、圣地亚哥、旧金山
- 🛣️ 一号公路自驾：世界最美海岸公路体验
- 🎢 主题公园：迪士尼、环球影城、海洋世界

## ✨ 核心特性

### 🏠 首页展示
- **英雄区块**：引人入胜的项目介绍和统计数据
- **行程亮点**：四大旅游阶段概览
- **路线预览**：交互式加州地图和路线展示
- **预算预览**：费用分布饼图和关键数据

### 📅 行程规划
- **时间线设计**：15天详细行程安排
- **阶段筛选**：按旅游阶段分组查看
- **互动展开**：点击查看每日详细活动
- **实用信息**：住宿、交通、贴士一目了然

### 🏛️ 目的地指南
- **城市探索**：洛杉矶、圣地亚哥、旧金山详细介绍
- **一号公路**：太平洋海岸线精选景点
- **智能筛选**：按类型（城市/自然）筛选
- **深度信息**：景点介绍、门票价格、游览建议

### 💰 预算规划器
- **实时计算**：互动式预算调整
- **费用分类**：机票、住宿、餐饮、门票详细分解
- **省钱贴士**：实用的预算优化建议
- **汇率显示**：人民币/美元双币种显示

### 🎒 后勤指南
- **住宿推荐**：4个城市的精选酒店信息
- **租车服务**：详细的租车配置和注意事项
- **打包清单**：分门别类的行李准备清单
- **实用贴士**：旅行中的重要提醒

## 🎨 设计理念

### 瑞士极简风格
- **网格系统**：严格的12列网格布局
- **字体层次**：清晰的标题和正文层级
- **色彩方案**：加州橙色(#FF5722)为主色调
- **留白艺术**：充分的留白提升阅读体验

### 交互设计
- **渐进式披露**：信息分层展示，避免认知负担
- **即时反馈**：按钮状态、hover效果等微交互
- **可访问性**：符合WCAG标准的无障碍设计
- **性能优化**：代码分割和懒加载

## 💻 本地开发

### 环境要求
- Node.js 16+ 
- pnpm (推荐) 或 npm

### 安装步骤
```bash
# 克隆项目
git clone https://github.com/MrColder666/california-family-trip-2027.git

# 进入项目目录
cd california-family-trip-2027/california-trip

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

### 开发命令
```bash
pnpm dev      # 启动开发服务器
pnpm build    # 构建生产版本
pnpm preview  # 预览生产构建
pnpm lint     # 代码检查
```
